Biến trong javascript : là 1 đối tượng cụ thể để lưu trữ thông tin

Biến trong js khai báo bằng lệnh var tên_biến;

Gán giá trị cho biến mình var tên_biến = 'giá trị'; trong TH vừa khai báo vừa gán

khai báo biến rồi thì tên_biến = 'giá trị';

trong js toán tử + có thể là toán tử nối chuỗi

khi mình nhiều chuỗi muốn nối với nhau : chuỗi 1 + chuỗi 2 + chuỗi 3 ( '3' + '1' => '31')

khi cộng 2 số với nhau thì + sẽ là phép cộng (3 + 1 = 4)

chuỗi + 1 số => dấu + sẽ là phép nối chuỗi
