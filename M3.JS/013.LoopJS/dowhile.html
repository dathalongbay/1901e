Vòng lặp do while

    do {
        // code block to be executed
        // code thực thi
    }
    while (điều kiện);

Vòng lặp while chỉ chạy khi điều kiện trong ngoặc đúng

Vòng lặp do ... while sẽ chạy khi điều kiện trong ngoặc đúng nhưng khi điều kiện không đúng thì nó chỉ chạy 1 lần đầu tiên và duy nhất
