Function JS

A JavaScript function is a block of code designed to perform a particular task.

Hàm trong js là khối các câu lệnh được tạo ra để thực hiện 1 nhiệm vụ cụ thể

A JavaScript function is executed when "something" invokes it (calls it).

Hàm trong js được chạy khi mà nó được gọi

Cú pháp hàm : function tên_hàm(arg1, arg2) { // code execute }

arg1 và arg2 nó tham số được truyền vào trong hàm

Gọi hàm : tên_hàm(x, y)
