Thay đổi css bằng js

Changing the style of an HTML element, is a variant of changing an HTML attribute:

document.getElementById('target').style.fontSize => lấy font size của id target ( 18px ) => parseInt(18px) => 18


var current_font_size = parseInt(document.getElementById('target').style.fontSize);

18 + 2 = 20


current_font_size = current_font_size + 2;

20 + 'px' = '20px'


var current_font_size_text = current_font_size + 'px';

gán ngược lại font size cho id target 22px => id target


document.getElementById('target').style.fontSize = current_font_size_text;
