Jquery là thư viện của js. Cho phép mình làm việc với js đơn giản và dễ dàng hơn.

JS còn angularjs , backbonejs, knockout js, coffee script ...

Bộ chọn ( selector ) trong jquery == bộ chọn ( selector ) css

ID CSS JQuery Chú thích
1 h1 $('h1') selector tên thẻ
2 #id $('#id') selector id
3 .class $('.class') selector class
4 .classA .classB $('.classA .classB') selector class lồng class
4 .classA > .classB $('.classA > .classB') selector con trực tiếp
