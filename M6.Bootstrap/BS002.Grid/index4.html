CSS Responsive Grid Class

  1. col-* áp dụng màn hinh mobile ( từ 0 đến 575px ) ví dụ col-1, col-2, col-3 ... col-12
  2. col-sm-* áp dụng màn hinh máy tính bảng như ipad ( từ 576px đến 767px ) ví dụ col-sm-1,col-sm-2, col-sm-3 ... col-sm-12
  3. col-md-* áp dụng màn hinh laptop ( từ 768px đến 991px ) ví dụ col-md-1, col-sm-2, col-md-3 ... col-md-12
  4. col-lg-* áp dụng cho màn hình PC ( từ 992px đến 1199px ) ví dụ col-lg-1, col-lg-2 ... col-lg-12
  5. col-xl-* áp dụng cho màn hình siêu lớn ( từ 1200px đến vô cực ) ví dụ col-xl-1, col-xl-2 ... col-xl-12
