Css display:inline-block

inline-block là sự kết hợp inline block

Chú ý : các thẻ inline như span , a , img thì đặt width và height trong css không có tác dụng

Con lai của block và inline

Nó sẽ lấy đặc tính hiển thị trên 1 hàng của inline

Nó sẽ lấy đặt tính đặt được width và height trong css của block

Khi nào cần sử dụng display:inline-block

cần đặt width và height cho các thẻ inline mà không muốn làm mất đặc tính hiển thị trên 1 hàng của các thẻ inline như span , a hay img
